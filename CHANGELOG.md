# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-06-28

### Added

- Monaco-powered JSON editor.
- Background worker parsing for large JSON payloads.
- Lazy tree rendering and 100-item array chunking.
- Expand-all guardrails for very large trees.
- README performance benchmark and release documentation.

### Changed

- Refreshed app and repository copy around performance and large JSON support.
- Improved navbar overlay layout and stacking above Monaco editor widgets.
